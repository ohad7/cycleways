How automatic direction proposals work
Exact reverse
Mechanically reverses every edge. Offered only when the complete reverse is permitted.
Roundabout reverse repair
Keeps the route unchanged except for replacing a backward roundabout arc with the permitted arc between the same entry and exit.
Authoring revision
Imports a newly recalculated base-edge mapping while protecting current or manually edited directional work.
Mapping coverage review
An automatic path may be directionally valid but include too much of a boundary edge. Inspect it with Add/remove edges; finishing that edit records your exact path choice.
