# App-Owned Approach ‚Äî Implementation Plan

**Date:** 2026-07-09

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship app-owned approach navigation in phases: first the editor data + confidence tooling, then runtime approach ownership with confidence-gated external handoff.

**Architecture:** Pure, node-tested core modules in `@cycleways/core` (`connectorFeatures`, `connectorConfidence`, `connectorEvaluate`) are reused identically by the editor and the app runtime. The editor labels connector routes into an append-only JSONL dataset via server endpoints. Runtime Phase 3 guides every successfully routed connector inside the product boundary and otherwise uses `too-far`; the approach leg switches to the existing main-route tracker at the seam instead of introducing negative-progress route tracking.

**Tech Stack:** Node ESM (`.mjs` tests run with `node tests/<file>.mjs`), the `@cycleways/core` workspace package, the editor's plain-Node `http` server (`editor/server.mjs`) and vanilla-JS client (`editor/editor.js`) with Mapbox GL layers, `node:crypto` for strategy hashing.

## Global Constraints

- **No map-data edits.** Never write `data/map-source.geojson`, `public-data/`, or any base-graph asset. The label dataset is a *new, separate* artifact.
- **`data/connector-eval/` is non-pipeline.** It must never enter the Build‚ÜíPromote flow. Its files (`labels.jsonl`, `strategies.json`) are **committed to git** as versioned ground truth ‚Äî but only via these editor endpoints, never regenerated by Build.
- **Shared feature/confidence logic is single-source.** `connectorFeatures` / `connectorConfidence` / `connectorEvaluate` live in `@cycleways/core` and are the only implementations; the editor imports them (Phase 3's app runtime will import the same modules). No reimplementation in the editor.
- **Features come from `previewBaseRoute().edgeCosts`.** Each traversal diagnostic already carries `routeClass`, `roadType`, `cyclewaysSegmentIds`, and per-traversal `distanceMeters`. Do not add a new per-edge descriptor to `route-manager.js`.
- **Snap failures are classified before quality math.** Persistent `snap-failed` after a diagnostic `snap:any` retry is off-network and excluded from reachable quality. `snap-ineligible` means the point is on the base graph but the current strategy cannot snap to it; count it as reachable-but-failed.
- **Labels are append-only events, not raw evaluation rows.** Calibration/evaluation must reduce the JSONL stream to the latest record per `(routeSlug, routeStart, origin, strategyHash, featureVersion)` before scoring, so relabeling does not double-count stale verdicts.
- **Successful routing owns guidance.** The router already enforces edge and
  access policy. `tooFarRadiusMeters: 10000` remains the product boundary;
  detour, route length, and accepted edge metadata do not downgrade a result.
- **Phase 3 v1 avoids negative progress.** Do not prepend the connector into the main route tracker yet. Track/render the connector as a separate approach leg while status is `approaching`, then switch to the existing main route when the rider acquires the seam/start. A true combined route with negative progress can be a later enhancement.
- **Tests are added to the `test` npm script** (the long `&&` chain in `package.json`). Run an individual test with `node tests/<file>.mjs`.
- **CW membership** is detected by `hasCyclewaysNetworkMembership(edge)` (already in `connectorCostModel.js`): truthy `cwSegmentIds`/`cyclewaysSegmentIds`. In `edgeCosts` diagnostics the field is `cyclewaysSegmentIds`.

```js
import { getDistance } from "../utils/distance.js";

// Worst-to-best ordering of connector edge classes for a "how sketchy is this
// path" signal. cw-owned edges are treated as the best class (cw_network).
export const CONNECTOR_CLASS_RANK = {
  cw_network: 0,
  road: 1,
  local_road: 2,
  cycle: 3,
  path_track: 4,
  manual: 5,
  other: 6,
};

function edgeClass(edge) {
  const cwOwned =
    Array.isArray(edge.cyclewaysSegmentIds) && edge.cyclewaysSegmentIds.length > 0;
  if (cwOwned) return "cw_network";
  if (edge.routeClass === "road" || edge.roadType === "road") return "road";
  const rc = edge.routeClass;
  return rc != null && rc in CONNECTOR_CLASS_RANK ? rc : "other";
}

export function computeConnectorFeatures(preview, { origin, routeStart } = {}) {
  const straightLineMeters =
    origin && routeStart ? getDistance(origin, routeStart) : 0;
  if (!preview || preview.failure) {
    return {
      snapOk: false,
      failure: preview ? preview.failure : "no-preview",
      straightLineMeters,
      routedMeters: 0,
      detourRatio: Infinity,
      cwNetworkFraction: 0,
      worstRouteClass: null,
      edgeCount: 0,
    };
  }
  const edges = Array.isArray(preview.edgeCosts) ? preview.edgeCosts : [];
  const routedMeters = Number(preview.distanceMeters) || 0;
  let cwMeters = 0;
  let worstRank = -1;
  let worstClass = null;
  for (const edge of edges) {
    const cls = edgeClass(edge);
    const d = Number(edge.distanceMeters) || 0;
    if (cls === "cw_network") cwMeters += d;
    const rank = CONNECTOR_CLASS_RANK[cls] ?? CONNECTOR_CLASS_RANK.other;
    if (rank > worstRank) {
      worstRank = rank;
      worstClass = cls;
    }
  }
  return {
    snapOk: true,
    failure: null,
    straightLineMeters,
    routedMeters,
    detourRatio:
      straightLineMeters > 1
        ? routedMeters / straightLineMeters
        : routedMeters > 0
        ? Infinity
        : 1,
    cwNetworkFraction: routedMeters > 0 ? cwMeters / routedMeters : 0,
    worstRouteClass: worstClass,
    edgeCount: edges.length,
  };
}
```

export const DEFAULT_CONNECTOR_THRESHOLDS = {
  tooFarRadiusMeters: 10000,
};

export function classifyConnector(features, thresholds = DEFAULT_CONNECTOR_THRESHOLDS) {
  if (!features || !features.snapOk) {
    return { tier: "too-far", handoffSuggested: true, reasons: ["unreachable"] };
  }
  if (features.straightLineMeters > thresholds.tooFarRadiusMeters) {
    return { tier: "too-far", handoffSuggested: true, reasons: ["beyond-too-far-radius"] };
  }
  return { tier: "guide", handoffSuggested: false, reasons: [] };
}
```

import { classifyConnector } from "./connectorConfidence.js";

// Score a set of human-labeled connectors against a threshold set. A "valid"
// label should be guided; an "unacceptable" label should not; "borderline" is
// counted for visibility but excluded from the precision/recall rates.
export function evaluateThresholds(labels, thresholds) {
  const counts = {
    valid: { guide: 0, other: 0 },
    unacceptable: { guide: 0, other: 0 },
    borderline: { guide: 0, other: 0 },
    total: 0,
  };
  for (const rec of labels || []) {
    if (!rec || !counts[rec.verdict]) continue;
    const { tier } = classifyConnector(rec.features, thresholds);
    counts[rec.verdict][tier === "guide" ? "guide" : "other"] += 1;
    counts.total += 1;
  }
  const validTotal = counts.valid.guide + counts.valid.other;
  const invalidTotal = counts.unacceptable.guide + counts.unacceptable.other;
  return {
    counts,
    validGuideRate: validTotal ? counts.valid.guide / validTotal : null,
    invalidGuideRate: invalidTotal ? counts.unacceptable.guide / invalidTotal : null,
  };
}
```

```js
import { appendFile, readFile, writeFile, mkdir } from "node:fs/promises";
import { dirname } from "node:path";
import { createHash } from "node:crypto";

const VERDICTS = new Set(["valid", "unacceptable", "borderline"]);

function isLatLng(p) {
  return p && Number.isFinite(Number(p.lat)) && Number.isFinite(Number(p.lng));
}

export function validateLabelRecord(rec) {
  if (!rec || typeof rec !== "object") return "record object required";
  if (!isLatLng(rec.routeStart)) return "routeStart {lat,lng} required";
  if (!isLatLng(rec.origin)) return "origin {lat,lng} required";
  if (!VERDICTS.has(rec.verdict)) return "verdict must be valid|unacceptable|borderline";
  if (!rec.features || typeof rec.features !== "object") return "features object required";
  return null;
}

// Deterministic JSON with sorted keys so a hash is independent of key order.
function stableStringify(value) {
  if (value === null || typeof value !== "object") return JSON.stringify(value);
  if (Array.isArray(value)) return `[${value.map(stableStringify).join(",")}]`;
  const keys = Object.keys(value).sort();
  return `{${keys.map((k) => `${JSON.stringify(k)}:${stableStringify(value[k])}`).join(",")}}`;
}

export function hashStrategy(strategy) {
  const hex = createHash("sha256").update(stableStringify(strategy)).digest("hex");
  return `sha256:${hex}`;
}

export async function appendLabel(path, rec) {
  const error = validateLabelRecord(rec);
  if (error) {
    const err = new Error(error);
    err.status = 400;
    throw err;
  }
  const record = { ...rec, ts: new Date().toISOString() };
  await mkdir(dirname(path), { recursive: true });
  await appendFile(path, `${JSON.stringify(record)}\n`, "utf8");
  return record;
}

export async function readLabels(path) {
  let text;
  try {
    text = await readFile(path, "utf8");
  } catch (e) {
    if (e.code === "ENOENT") return [];
    throw e;
  }
  return text.split("\n").filter(Boolean).map((line) => JSON.parse(line));
}

export async function upsertStrategy(path, strategy) {
  const hash = hashStrategy(strategy);
  let existing = {};
  try {
    existing = JSON.parse(await readFile(path, "utf8"));
  } catch (e) {
    if (e.code !== "ENOENT") throw e;
  }
  if (!existing[hash]) {
    existing[hash] = strategy;
    await mkdir(dirname(path), { recursive: true });
    await writeFile(path, `${JSON.stringify(existing, null, 2)}\n`, "utf8");
  }
  return hash;
}
```
